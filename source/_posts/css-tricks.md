### css (cascading style sheets)

#### color inherit

  颜色要多用继承，这样可以统一修改

#### inline-block

  最大的好处就是可以不写宽度，和 inline-flex 一样在行内具有块元素的属性
  用 vertical-align 或者 font-size 处理默认的高度

#### pseudo class

  唯一需要指定的就是 content
